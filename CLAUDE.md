## 평가 기준 (50:50)
- **50% 피어 투자**: 다른 팀들이 데모 체험 후 투자 칩 배분
- **50% 전문가 심사**: 문제 정의, 시장성, 설득력, 피칭

## 산출물
1. 1페이지 기획서 (문제 정의 / 해결 방법 / Upstage 활용 이유)
2. 발표 장표 (PDF)
3. GitHub 레포 (README + 코드 + 기획서 + 발표자료)
4. 실제 작동하는 데모 (현장 시연 필수)

## Upstage API
- **콘솔**: console.upstage.ai
- **API 문서**: console.upstage.ai/docs/for-agents
- **크레딧 코드**: UPWAVE-YONSEI (연세대) / UPWAVE-KOREA (고려대) → $80 크레딧

### 사용 가능한 제품
| 제품 | 설명 |
|------|------|
| **Document Parse** | PDF/스캔 이미지 → HTML/Markdown 변환. 업계 1위 정확도 |
| **Information Extract** | 문서 + JSON 스키마 → 구조화된 데이터 추출. GPT-4.1보다 정확 |
| **Document Classify** | 문서 자동 분류 (학습 불필요) |
| **Solar LLM** | 한국어 특화 LLM (Solar Pro 3: 102B MoE, 128K context) |
| **Upstage Studio** | 노코드 문서 워크플로우 빌더 |

## 현재 검토 중인 아이디어
1. **진단서/소견서 번역기** — 의학 용어를 일반인 언어로 변환 (복약 체크 기능은 의료법 이슈로 제외)
2. **시험 족보 분석 + 유사 문제 생성기** — 기출 PDF 파싱 → 출제 패턴 분석 → 유사 문제 생성 (소형 학원/과외 타깃)
